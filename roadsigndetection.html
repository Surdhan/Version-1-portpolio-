AUTOMATIC ROAD SIGN DETECTION AND VEHICLE CONTROL USING USING NEURAL NETWORKS

This project leverages Raspberry Pi and advanced image processing to create a robust traffic sign detection system. Overcoming challenges in low light, adverse weather, and distorted sign conditions, it employs adaptive algorithms and machine learning for real-time recognition. At present human beings are facing many accidents during the road ways transportation. At the same time they lose their life and valuable properties in these accidents. To avoid these problems this system designed with the help of Raspberry Pi. The digital image processing plays important role in the sign capturing and detection system. Specifically, we created a road sign recognition system based on an embedded system that reads and recognises road signs requirements and difficulties behind implementing real time base system with embedded system and how to deal with numbers using image processing techniques based on shape and dimension analysis. This project also shows the techniques used for classification and recognition. In this project lightweight techniques were mainly used due to limitation of real time-based applications and raspberry Pi capabilities. Web Camera is used to capture the road signs with image enhancement techniques the embedded systems small computing platform studies the characteristics of speed and road signs. In daylight vision time to take the shape analysis for recognising the sign using Image Processing. Raspberry Pi is the main target for the implementation as it provides an interface between sensors, database and image processing results, while also performing functions to manipulate peripheral units
